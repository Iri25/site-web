# site-web

## Setup 

1. Backend
  -   Run `npm install` to install all project dependencies
  -   Run `npx nodemon src/server.js | npx run dev | npm run dev` to start-up the server 

2. Frontend
   - Run `npm install` to install all project dependencies
   - Run `npm run dev` to run project on the Web
